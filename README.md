A simple and responsive Companies Directory built using React, Vite, and TailwindCSS.  
The app displays company data from a mock JSON API and supports searching, filtering, and pagination.

Live Demo: https://companies-directorys.netlify.app/

---

## Features
- Search companies by name  
- Filter by **location**  
- Filter by **industry**  
- Clear filters button  
- Pagination (Next/Prev)  
- Loading spinner  
- Responsive UI  

---

## Tech Stack
- React.js (Vite)  
- TailwindCSS  
- React Hooks  
- react-spinners  
- Netlify Deployment  

---

## Setup & Run

```bash

````

